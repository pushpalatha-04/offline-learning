Offline Education App

A React-based offline education application designed for school students to learn anytime, anywhere — no internet connection required.

 Features

Quizzes & Tests** — Interactive assessments to reinforce learning
Progress Tracking** — Monitor student performance and growth over time

 Tech Stack

Frontend: React.js
Styling:CSS
Storage:LocalStorage / IndexedDB (offline data persistence)

 Installation

1. Clone the repository
   git clone https://github.com/pushpalatha-04/offline-learning.git
   cd offline-education-app
  

2.Install dependencies
   npm install
  

3. Start the development server
   npm start

4. Open your browser and go to `http://localhost:3000`

 How to Use

1. Launch the app in your browser
2. Browse available lessons or subjects
3. Take quizzes to test your knowledge
4. View your progress on the Progress Tracking page
